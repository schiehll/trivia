# Testing

We are using <Link to="jest">Jest</Link> to run our tests.

## test

Just run the following command to see the tests passing:

```bash
$ yarn test
```

## watch

If you need to run it in `watch` mode, just do:

```bash
$ yarn test --watch
```

## coverage

If you want to see the test coverage:

```bash
$ yarn test --coverage
```
